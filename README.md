# Electronics
Приложение для курсовой работы
## Запуск приложения
1. для установки зависимостей
```
npm i
```
2. для запуска веб интерфейса
```
npm run dev
```
3. для запуска сервера базы данных
```
npm run db
```
4. веб сервер - `http://localhost:5173/`, сервер БД - `http://localhost:3000/`
5. данные для входа
Логин
```
manager_1
```
Пароль
```
qwerty123
```
